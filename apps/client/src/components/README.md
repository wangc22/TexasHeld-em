# components

UI components organised by domain. Each subdirectory is a self-contained module.

| Directory | Contents |
|---|---|
| `Actions/` | Player action buttons (fold, check, call, raise, all-in) |
| `Audio/` | Background music player |
| `Card/` | Single playing card renderer |
| `Game/` | Game-level overlay panels (bot thoughts, hand history, leaderboard, quick chat, add-bot modal) |
| `Player/` | Player seat and chip stack display |
| `Table/` | Poker table layout, community cards, bet display |
| `Tutorial/` | How-to-play modal |
| `Voice/` | WebRTC voice chat controls |
